# Dutybox 개인정보처리방침

**시행일**: 2026년 5월 5일
**최종 갱신일**: 2026년 5월 5일

> 이 문서는 모바일 애플리케이션 **Dutybox** (이하 "본 서비스" 또는 "앱")의 개인정보 수집·이용·보관·파기 정책을 안내합니다. 본 서비스는 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」(이하 "정보통신망법")을 준수합니다.

---

## 1. 수집하는 개인정보 항목

본 서비스는 회원 가입과 서비스 이용 과정에서 다음과 같은 개인정보를 수집합니다.

### 1-1. 회원가입 시 수집

| 구분 | 항목 | 수집 방법 |
|---|---|---|
| 카카오 간편가입 | 이메일, 닉네임, 프로필 사진 URL, 카카오 회원번호(고유 식별자) | 카카오 OAuth 동의 항목 |
| 이메일 가입 | 이메일, 비밀번호(암호화 저장) | 이용자 직접 입력 |
| 공통 | 닉네임, 소속 회사명(선택) | 이용자 직접 입력 |

### 1-2. 서비스 이용 과정에서 자동 수집·생성

| 항목 | 설명 |
|---|---|
| 근무 스케줄 | 날짜·근무 유형(근무/휴무/대기 등)·근무 라벨·시작·종료 시간·메모 |
| 친구 관계 정보 | 친구 사용자 ID, 그룹명·그룹 멤버십, 초대 코드 |
| 응원·메세지 | 이모지, 100자 이내 텍스트 메시지, 발신자·수신자, 작성·수정 시각 |
| 버그리포트 | 이용자가 첨부한 스크린샷(최대 3장), 이용자가 작성한 코멘트 |
| 디바이스/접속 정보 | 운영체제 종류·버전, 앱 버전, 접속 시각, 네트워크 연결 상태 |
| 자동 부여 식별자 | 본 서비스 내부 회원 식별자(UUID), 초대 코드 |

### 1-3. 수집하지 않는 항목

- **OCR 업로드 이미지**: 이용자가 스케줄표 자동 입력 기능을 사용할 때 업로드한 이미지는 AI 분석 직후 즉시 폐기되며, 본 서비스 서버에 저장되지 않습니다. 분석 결과(스케줄 텍스트 정보)만 저장됩니다.
- **위치 정보, 연락처, 통화 기록, SMS, 단말 식별번호(IMEI/광고 ID)**: 수집하지 않습니다.

## 3. 보유 및 이용기간

| 항목 | 보유기간 |
|---|---|
| 회원 정보 및 관련 모든 데이터 | **회원 탈퇴 시 즉시 영구 삭제** |
| 버그리포트 첨부 이미지 | **업로드 후 7일** (자동 만료) |
| 버그리포트 메일 본문(운영자 메일함) | 운영자 메일 정책에 따라 별도 보관 |
| 단말 내 캐시(앱 내부) | 앱 삭제 시 자동 삭제 |

회원 탈퇴 요청 시 본 서비스의 모든 데이터(스케줄, 친구 관계, 응원/메세지, 프로필 등)는 데이터베이스 외래키 cascade 규칙에 따라 자동으로 영구 삭제됩니다. 백업본은 별도로 보관하지 않습니다.

다만, 관계 법령(전자상거래법·통신비밀보호법 등)에 따라 보존이 의무화된 경우에는 해당 법령이 정한 기간 동안 분리 보관 후 파기합니다.

## 5. 개인정보의 파기 절차 및 방법

### 5-1. 파기 절차

이용자가 회원 탈퇴를 요청하거나 보유기간이 종료된 경우, 본 서비스는 즉시 해당 개인정보를 데이터베이스에서 영구 삭제합니다.

### 5-2. 파기 방법

- **전자적 파일**: 복원이 불가능하도록 데이터베이스 레코드 영구 삭제(DELETE) 처리
- **출력물**: 본 서비스는 이용자의 개인정보를 출력물로 보관하지 않습니다.

## 6. 이용자 및 법정대리인의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

| 권리 | 행사 방법 |
|---|---|
| 개인정보 열람 | 앱 내 설정 → 프로필 수정 |
| 개인정보 수정 | 앱 내 설정 → 프로필 수정 |
| 개인정보 삭제 (회원 탈퇴) | 앱 내 설정 → 우측 하단 "계정 탈퇴" 또는 본 처리방침 하단 운영자 메일로 요청 |
| 개인정보 처리 동의 철회 | 회원 탈퇴 시 자동으로 모든 동의가 철회됩니다 |

회원 탈퇴는 앱 내에서 즉시 처리되며, 별도의 승인 절차 없이 모든 데이터가 영구 삭제됩니다.

## 7. 개인정보의 안전성 확보 조치

본 서비스는 이용자의 개인정보를 안전하게 보관하기 위해 다음 조치를 취하고 있습니다.

- **암호화 통신**: 이용자 단말과 서버 간 모든 통신은 TLS(HTTPS)로 암호화됩니다.
- **비밀번호 암호화**: 이메일 가입 비밀번호는 단방향 해시(bcrypt)로 저장되어 운영자도 평문을 조회할 수 없습니다.
- **접근 통제 (Row-Level Security)**: 데이터베이스 단계에서 본인 또는 친구 관계가 확인된 경우에만 데이터 조회·수정이 가능하도록 정책을 적용합니다.
- **인증 토큰 관리**: 로그인 세션은 단기 만료되는 JWT 토큰으로 관리되며, 갱신 토큰은 단말에 안전하게 저장됩니다.
- **첨부파일 만료 URL**: 버그리포트 첨부 이미지는 7일 만료 서명 URL로만 접근 가능합니다.
- **최소 수집 원칙**: 서비스 제공에 반드시 필요한 항목만 수집하며, 위치·연락처·광고 식별자는 수집하지 않습니다.
