# Dutybox 계정 삭제 안내

**시행일**: 2026년 5월 5일

> 본 페이지는 Google Play Store 정책에 따라 앱 외부에서도 계정 삭제를 요청할 수 있도록 안내하는 공식 안내 페이지입니다.

---

## 1. 앱 내에서 계정을 삭제하는 방법 (권장)

앱이 설치되어 있다면 다음 경로로 즉시 영구 삭제할 수 있습니다.

1. **Dutybox** 앱 실행
2. 우측 상단 **⚙️ 설정** 아이콘 → 우측 드로어 열기
3. 화면 **우측 하단**의 작은 빨간색 **"계정 탈퇴"** 텍스트 버튼 탭
4. 안내 다이얼로그 확인 → **"정말 삭제할까요?"** 한 번 더 확인
5. 잠시 후 **"이용해주셔서 감사합니다."** 메세지가 표시되면 삭제 완료. 자동으로 로그인 화면으로 돌아갑니다.

> 삭제는 **즉시 영구적으로** 진행되며, 이후 동일 계정으로 다시 로그인하더라도 이전 데이터는 복구되지 않습니다.

---

## 2. 앱이 설치되어 있지 않은 경우 — 이메일로 삭제 요청

앱을 이미 삭제했거나 단말 접근이 어려운 경우, 아래 이메일로 계정 삭제를 요청해 주세요.

**삭제 요청 이메일**: _(추후 기재 — 예: support@naeilgeunmu.app)_

요청 시 본인 확인을 위해 다음 정보를 함께 보내 주세요.

- 가입 시 사용한 이메일 주소 또는 카카오 닉네임
- 가입 방식 (카카오 / Apple / 이메일)
- 본인 확인을 위한 추가 정보(필요 시 운영자가 별도 안내)

운영자는 요청을 수신한 날로부터 **영업일 기준 7일 이내**에 본인 확인 후 계정을 영구 삭제하고, 처리 결과를 회신해 드립니다.

---

## 3. 삭제되는 데이터

회원 탈퇴 시 다음 모든 데이터가 **즉시 영구 삭제**됩니다.

- 계정 정보 (이메일, 닉네임, 프로필 사진 URL, 소속 회사명 등)
- 모든 근무 스케줄 (날짜·근무 라벨·시간·메모)
- 친구 관계 (친구 목록, 그룹 멤버십, 초대 코드)
- 응원 및 메세지 (보낸/받은 모든 이모지·텍스트 메세지)
- 버그리포트 첨부 이미지 (Storage)
- 단말 내 캐시 (자동 삭제)

---

## 4. 삭제되지 않는 데이터 (관계 법령 의무 보관)

본 서비스는 관계 법령에 따라 보존이 의무화된 데이터가 발생하는 경우, 해당 법령이 정한 기간 동안만 분리 보관 후 파기합니다. **현재 본 서비스는 결제·전자상거래 기능을 제공하지 않으므로, 의무 보관 데이터가 없습니다**.

---

## 5. 카카오 / Apple / Google 연동 해제

본 서비스의 회원 탈퇴는 본 서비스 내부 데이터만 삭제합니다. 카카오·Apple·Google 등 외부 OAuth 제공자에 저장된 동의 정보를 함께 해제하려면 다음 경로를 이용해 주세요.

| 제공자 | 연동 해제 방법 |
|---|---|
| 카카오 | 카카오톡 앱 → 더보기 → 설정 → 카카오계정 → 연결된 서비스 관리 → "Dutybox" 연결 끊기 |
| Apple | 설정 앱 → Apple ID → 로그인 및 보안 → Apple로 로그인 사용 중인 앱 → "Dutybox" → 사용 중단 |
| Google | _(향후 도입 시 안내 예정)_ |
